# spring-start-arc

本项目用于快速开始一个以 spring boot 为基础的项目框架

## 集成的框架

1. spring boot
2. mybatis && spring mybatis && mybatis generator
3. flyway 
4. l-spring-log

## 注意事项


## 使用方式

1. clone 项目到本地
2. git submodule init --update
3. cd libs/l-spring-log
4. mvn clean install
5. cd /path/to/project
6. mvn clean install
